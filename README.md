# on-jiuyou-lab

## 项目简介

`on-jiuyou-lab` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
这里存放的页面不绑定任何特定域名或网站，仅作为静态页面的集中存储与版本管理。

本仓库适用于以下场景：
- 保存独立的 HTML 页面快照
- 对页面进行版本迭代与记录
- 提供页面原始文件的公开访问

## 目录说明

```
.
├── pages/              # 存放所有独立 HTML 页面文件
├── assets/             # 页面引用的公共资源（如 CSS、JS、图片等）
├── index.html          # 仓库根目录索引页面（可选）
└── README.md           # 本说明文件
```

- `pages/` 目录下每个子文件夹对应一个独立页面项目，内含其 HTML 及专属资源。
- `assets/` 存放多个页面可能共用的资源，避免重复。
- 各页面之间无依赖关系，可独立访问。

## 页面归档说明

- 每个页面归档包含完整的 HTML 结构及相关资源，确保离线或在线均可正常浏览。
- 归档的页面可能来自不同时期、不同用途，风格与功能各异，统一归类于此。
- 页面版本通过 Git 提交历史管理，可追溯每次变更。

## 维护说明

- 本仓库为静态内容仓库，不涉及后端服务或动态数据。
- 页面更新时，请保持原有目录结构，避免影响已有链接。
- 如有页面废弃或移除，建议在提交信息中注明原因。
- 欢迎提交 Issue 讨论页面内容或结构改进建议。

## 使用方式

你可以通过以下方式使用本仓库内容：

1. **直接浏览**：访问仓库页面，在 `pages/` 目录下找到感兴趣的 HTML 文件并打开。
2. **克隆到本地**：`git clone https://github.com/your-username/on-jiuyou-lab.git` 后，在本地用浏览器打开 HTML 文件。
3. **部署到静态托管服务**：将整个仓库部署到任意静态文件服务器，即可在线访问所有页面。
